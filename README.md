Collection of tools to allow for converting Windows applications that use OpenGL or Vulkan to NSP files for use on a Nentendo Switch

This is probably the most autistic and brain damaged organization on GitHub, but that's partially the goal.

The project will likely be deamed complete once a real game can be played on it. Depending on the scope, the game is likely to be the 25th aniversary version of Half Life (but that depends on many factors of course).

In the future, I would like to add support for DirectX (possibly through DXVK?), and the game for testing that will likely be Half Life 2.
